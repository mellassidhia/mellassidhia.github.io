# 🌐 Portfolio

This repository contains my personal portfolio website.
It highlights my work, projects, and technical skills in a simple and responsive design.

---

## 🚀 Overview

The portfolio is designed to:

* Present who I am
* Showcase my projects
* Provide an easy way to explore my work
* Offer a clean and modern user experience

---

## 🛠️ Tech Stack

* HTML5
* CSS3

---

## 📂 Project Structure

```
portfolio/
│── index.html
│── css/
│   └── styles.css
└── assets/
   └── images/
```

---

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/mellassidhia/mellassidhia.github.io.git
```

Open the project:

```bash
cd mellassidhia.github.io
```

Then open `index.html` in your browser.
